경로 설명

'/' => 홈페이지 

'/page' => 프론트쪽 / 해당 ejs를 알려줌, render을 해주는 곳 

'/auth' => 백앤드쪽 / 로그인,로그아웃,회원가입 같은 역할을 해줌

'/adminbro' => 관리자 환경 / 사용자 학생증 확인 및 login allow , report 처리 등을 해줌

'/select' => 강의실 입실 / 사용자가 선택한 강의실을 보여주고 입실 및 퇴실 예약 기능이 들어있음

'/report' => 신고하기 / 해당 강의실을 사용했던 사용자를 신고하는 역할, 관리자 환경에서 처리 가능
